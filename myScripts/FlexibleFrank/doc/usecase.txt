・通常のバッファに通常のテキストとして開く

・編集禁止

・TabColserでの強制対象

・コマンド
　!Enter  : 選択行をedit（ファイル時のみ、単行実行可能）
　!Enter  : cd　  　　　（ディレクトリ時のみ、単行実行のみ）
　+!o     : open　  　　（ディレクトリ時のみ、単行実行のみ）
　+!Enter : フランクを開く
　F5      : 更新
　Space   : ポイント
　+Space  : ポイント取り消し
　h       : cd ../
　l       : cd -
　x       : 関連づけされたアプリケーションで開く
　c       : 選択行をカーソル階層にコピー
　m       : 選択行をカーソル階層に移動
　d       : 選択行をカーソル階層に削除
　r       : 選択行をリネームバッファで編集する（同階層のみかつファイルのみ選択時に有効）

・表示の更新
　+!Enter : 表示更新
　cd, lcdされたときにも表示更新される

・TAB
　２つめの作業ファイルとの移動
　１つしか開いていないときは横分割でもう一つ開く

・表示に用いる区切り文字
　windows : \
　mac     : /

・関連別バッファ
　作業ファイルは別バッファにより閉じられる
　以下の動作で閉じる
　　別バッファへの移動
　　vimからフォーカスを外す
　TabColserでの強制対象
　関連別バッファを閉じた後、作業ファイルを開く

・リネームバッファ
　別バッファで選択されたファイル名を表示
　保存するとファイル名が変更される

・折り畳みが可能　表示詳細は folding.txt 参照
　カーソル階層以下を畳む
　カーソル階層以下を開く
　カーソル階層以下を畳む（再帰）
　カーソル階層以下を開く（再帰）
