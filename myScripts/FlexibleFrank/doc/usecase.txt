・通常のバッファに通常のテキストとして開く

・編集禁止

・TabColserでの強制対象

・コマンド
- +!Enter : 　　　　フランクを開く
- e       : １　　　選択行をedit（ディレクトリポイントはスキップ、未ポイントはカーソル下）
- m       : １，２　cd　  　　　（ディレクトリ時のみ、ポイント無視）
- h       : １，２　cd ../
- l       : １，２　cd -
- o       : １，２　open　  　　（ディレクトリ時のみ、ポイント無視）
- F5      : １，２　更新
- p       : １　　　ポイント
- +p      : １　　　ポイント取り消し
- a       : １　　　関連づけされたアプリケーションで開く（ファイル時のみ、ポイント無視）
　c       : ２　　　選択行をカーソル階層にコピー
　m       : ２　　　選択行をカーソル階層に移動
　d       : ２　　　選択行をカーソル階層に削除
　r       : １　　　選択行をリネームバッファで編集する（同階層のみかつファイルのみ選択時に有効）

・表示の更新
　更新キーの入力
　cd, lcdされたときにも表示更新される

・TAB
　フランク２の作業ファイルとの移動
　フランク１しか開いていないときは横分割でもう一つ開く

・表示に用いる区切り文字
　windows : \
　mac     : /

・関連別バッファ
　作業ファイルは別バッファにより閉じられる
　以下の動作で閉じる
　　別バッファへの移動
　　vimからフォーカスを外す
　TabColserでの強制対象
　関連別バッファを閉じた後、作業ファイルを開く

・リネームバッファ
　別バッファで選択されたファイル名を表示
　保存するとファイル名が変更される

・折り畳みが可能　表示詳細は folding.txt 参照
　カーソル階層以下を畳む
　カーソル階層以下を開く
　カーソル階層以下を畳む（再帰）
　カーソル階層以下を開く（再帰）
