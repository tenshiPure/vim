・通常のバッファに通常のテキストとして開く

・編集禁止

・閉じるタイミング
　別バッファへの移動
　vimからフォーカスを外す
　TabColserでの強制対象

・移動コマンドは上下系のみ
　k  : １行上にカーソル移動
　j  : １行下にカーソル移動
　K  : 10行上にカーソル移動
　J  : 10行下にカーソル移動
　gk : １つ上行のディレクトリにカーソル移動
　gj : １つ下行のディレクトリにカーソル移動
　gK : 同じ階層の１つ上行のディレクトリにカーソル移動
　gJ : 同じ階層の１つ下行のディレクトリにカーソル移動
　gg : 先頭行へカーソル移動
　G  : 最終行へカーソル移動

・範囲指定
　V  : 行単位を範囲指定
　v, : カーソル階層以下を範囲指定

・行に対する操作
　Enter  : edit　　（ファイル時）
　!Enter : tabedit （ファイル時）
　Enter  : cd　  　（ディレクトリ時）
　!Enter : open　　（ディレクトリ時）

・行選択
　Space  : 行選択
　+Space : 行選択取り消し

・表示の更新
　+!Enter : 表示更新
　cd, lcdされたときにも表示更新される

・TAB
　２つめの作業ファイルとの移動
　１つしか開いていないときは横分割でもう一つ開く

・表示設定とソート設定
　o : 設定バッファを開く

・ファイル操作
　x : 関連づけされたアプリケーションで開く
　c : 選択行をカーソル階層にコピー
　m : 選択行をカーソル階層に移動
　d : 選択行をカーソル階層に削除
　r : 選択行をリネームバッファで編集する（同階層のみ選択時に有効）

・表示に用いる区切り文字
　windows : \
　mac     : /

・関連別バッファ
　作業ファイルは別バッファにより閉じられる
　以下の動作で閉じる
　　別バッファへの移動
　　vimからフォーカスを外す
　TabColserでの強制対象
　関連別バッファを閉じた後、作業ファイルを開く

・リネームバッファ
　別バッファで選択されたファイル名を表示
　保存するとファイル名が変更される

・折り畳みが可能　表示詳細は folding.txt 参照
　カーソル階層以下を畳む
　カーソル階層以下を開く
　カーソル階層以下を畳む（再帰）
　カーソル階層以下を開く（再帰）
